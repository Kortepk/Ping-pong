(Можно следовать скринам, они в папке "скрины для инструкции")

1) Запуск сервера (server.exe)
2) Появиться данные порта (айпишки для локалки другой)

3) Заходим в windows "Параметры сети и интернет"
4) Вкладка Ethernet → Настройка параметров адаптера → ПКМ по сети → состояние → сведения
5) Смотрим адрес IPv4 (192.168.0.71) (различается только последняя цифра)
6) Запоминаем

7) Запускаем main.exe
8) Если сервер запущен на этом же компе что и main, то пишем h
   Иначе пишем i нажимаем интер, далее попросит айпи, вводим айпи компа, где запущен сервер (192.168. ...)
9) Игроки должны подключаться по очереди. Для первого игрока порт: 53000, для второго: 53001
10) Если успешно, то запуститься окно с выбором цвета, выбираем цвет с помощью ползунков или кнопки на экране R (random)
11) Далее нажимаем на выбранный цвет (самый большой прямоугольник) или Enter

12) Ждём второго игрока (для игрока 2 шаги: 9-11)

13) При успешном подключении игроков появиться на экране игра

Управляется кнопками WASD
Управление из любой части экрана и приложения

При нажатие на кнопку Esc отключается игрок + сервер (т.е. для выхода), либо можно закрыть окна/сервер

Удачи в игре ☺ :-)
